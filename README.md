# termaudio: A Terminal Music Player

## How To Use

After installing termaudio, run it like so:

```
python termaudio PATHNAME ...
```

PATHNAME may be a file or a directory.

At the termaudio screen, move around with the following keys

- `q` - Quit
- `k` - Pause/Play
- `n` - Next In Oueue
- `l` - skip 5 seconds
- `j` - rewind 5 seconds
- `=` - volume up
- `-` - volume down
- `s` - shuffle
- `x` - sort
- `r` - repeat one
